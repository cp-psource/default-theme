== Description ==

Mit Standarddesign kannst Du ein benutzerdefiniertes Standarddesign für neue Webseiten in Deinem Netzwerk festlegen.

= Es ist einfach =

Rufe nach der Aktivierung Netzwerk-Admin > Einstellungen auf und wähle das Design aus, das auf allen neu erstellten Webseiten zuerst angezeigt werden soll.
